Blog 3: Glowing Button

For this weeks blog I wanted to see what other elements there were in CSS and HTML. So I decided to look at other aspects that we haven't seen yet in class. So, I decided to take a look at what could be done with buttons since they are a very important part of any website we go to. After looking for a while I found a way to get a buttons border to be high lighted and show differnt colors when hovered over.

There are a couple of key aspects to making this button work, there is the actual animation that takes place in the key frames to create the glowing effect with they background position moving from 0% to 100% after activation to give the idea that it is fading in. The next aspect of button that makes it work are the classes that pertain to glow on hover. The first is the actual base which sets up the actual size of the box along with the border that will actually be highlighted later on. The next is the before class which sets up the colors along with the actual movement of the border, calc is used to change the size of the button along with the opacity of the animation. The other methods are for the activation of the button the main gist of them are mostly so the black background disappears on activation and comes back after words.
