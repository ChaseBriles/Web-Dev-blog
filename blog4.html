CSS Filters

Filters in CSS are very interesting and extremly useful to create a more dynamic and interesting webpage. There are multiple ways to have filters effects added on to a image with CSS. The first one is that CSS comes onboard with the "Filter" property. With this there are multiple different aspects that you can add to the image like adding an amount of blur to distort the image, adding gray to the image, changing the color contrast on the image to make it more vibrant or dim, or even adding a drop shadow effect to the image. You can also combine these effects to create differnt types of like making to images look gooey using differnt values in contrast and blur to make them overlap and make them look like they a gooey or at least warped togeather.

The other way to get filters in css is to import them using SVG links. SVG filters can be extremly handy since they give a much wider range of what can be done with the filters. There are a lot more vairables to change using SVG like for blur you can also change the width, height, x positioning, and y positioning. However, since there are so many differnt browers and there are always issues with what is actually supported in CSS, SVG filters are no differnt. At this point only SVG element filters work in Firefox and Firefox for android, however all other browers support the normal filter properties that come along with CSS except for some versions of Internet Explorer.
